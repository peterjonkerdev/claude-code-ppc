# Lessons Learned

Corrections and rules captured during work. Each entry prevents a past mistake from recurring.

## How to use

After any correction, add an entry:
- **Date** — when it happened
- **Context** — what went wrong
- **Rule** — concrete rule to prevent recurrence

These supplement the self-improvement loop in CLAUDE.md. CLAUDE.md gets the short rule; this file gets the full context.
